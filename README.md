# 🚀 TriageAI Backend

An intelligent **FastAPI-based backend system** designed for emergency triage.
It uses **context pruning and rule-based reasoning** to provide fast and relevant medical action recommendations.

---

## 📌 Features

* ⚡ **FastAPI Backend** with high performance
* 🧠 **Intelligent Context Pruning**
* 🩺 **Basic Medical Reasoning Engine**
* 📊 Token usage tracking & pruning efficiency
* 🗂️ In-memory database for ultra-fast access
* 🔁 Response caching for repeated queries
* 🌐 REST API with automatic Swagger UI

---

## 🛠️ Tech Stack

* Python 3.10+
* FastAPI
* Uvicorn
* Pydantic
